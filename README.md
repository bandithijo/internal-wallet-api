# README: Internal Wallet API

## Ruby version
```
3.3.5
```

## Configuration
```
$ bundle install
```

## Database initialization
```
$ bin/rails db:create
$ bin/rails db:migrate
```

## Data seeds
```
$ bin/rails db:seed
```

## Run the test suite
```
$ bin/rails rspec
```

## Run the server
```
$ bin/rails s
```
